# 基于linux mint 22.3 安装后 安装 oxwm 

oxwm 仓库地址  git@github.com:syaofox/oxwm.git

# zig 安装

```bash
wget -q https://ziglang.org/download/0.15.2/zig-x86_64-linux-0.15.2.tar.xz -O /tmp/zig.tar.xz && sudo tar -xf /tmp/zig.tar.xz -C /opt/ && sudo ln -sf /opt/zig-x86_64-linux-0.15.2/zig /usr/local/bin/zig && zig version

```

# oxwm 依赖

```bash
sudo apt update
sudo apt install git libx11-dev libxft-dev libxinerama-dev libxrandr-dev libxcursor-dev libxcomposite-dev libxdamage-dev liblua5.4-dev
```



# 克隆并编译 oxwm
在终端中执行以下命令来获取源代码并构建项目：

# 克隆仓库
git clone git@github.com:syaofox/oxwm.git /tmp/oxwm
cd /tmp/oxwm

# 构建项目 (优化为 ReleaseSmall 以获得更小的体积)
sudo zig build -Doptimize=ReleaseSmall --prefix /usr


# 设置启动
通过显示管理器 (Mint 默认 LightDM)

创建一个 .desktop 文件以便在登录界面选择 oxwm：

创建文件：sudo nano /usr/share/xsessions/oxwm.desktop

粘贴以下内容：

```bash
[Desktop Entry]
Encoding=UTF-8
Name=oxwm
Comment=Dynamic window manager written in Zig
Exec=xxx # 填入当前 dotfiles/oxwm/oxwm-start.sh 的绝对路径
Icon=oxwm
Type=XSession
```
